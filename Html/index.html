Introduction to HTML

HTML - Hyper Text Markup Language.

HTML is a markup language which provides a set of tags suitable for making up webpages.

HTML is a hyper text language because it supports font styled text , pictures , graphics and animations .

Structure of the Html document

All HTML documents does follow some basics structure.

  • Body block

Attributes provide additional information about an element:

Common attributes:

  • src: Specifies source for images or media.
  • href: Specifies URL for links.
  • alt: Describes images for accessibility.

4. Semantic Elements

HTML5 introduced semantic elements to improve readability:

  • <header>: Defines the header section.
  • <footer>: Defines the footer section.
  • <article>: Represents independent content.
  • <section>: Groups related content.
